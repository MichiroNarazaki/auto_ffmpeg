# 概要

### 音が発生した時点から10秒間のみを抽出し、それらの動画を結合して1つの動画に編集します

### ffmpeg のインストールが必要です

mac -> brew install ffmpeg

# 使い方

0.  git clone --recursive https://github.com/MichiroNarazaki/auto_ffmpeg
1.  cd libft
2.  make
3.  cd ../
4.  vi main.sh
5.  write the origin movie path
6.  save (esc -> :wq -> enter)
7.  ./main.sh
8.  You can watch the movie you want.
